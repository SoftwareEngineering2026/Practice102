### Краткое описание проекта

`twitch-notifier` — это сервис, написанный на Go, предназначенный для отправки уведомлений о начале стримов Twitch в мессенджеры (например, Telegram).

Проект включает:

- работу с Twitch API
- конфигурационный модуль
- обработку событий стриминга
- отправку уведомлений
- CI-проверки и тесты
