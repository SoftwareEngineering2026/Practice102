# Отчёт о тестировании проекта Chinese Teacher API

## 1. Выбранный проект
- **Название**: Chinese Teacher API
- **Ссылка на репозиторий**: https://gitlab.com/ai-chinese-agent/Backend
- **Краткое описание**:
  FastAPI сервис для анализа произношения китайского языка. Принимает аудио с речью пользователя, распознает текст через Whisper, сравнивает с эталоном и дает обратную связь по ошибкам произношения и тонов.

## 3. Что было покрыто тестами
- **Модели данных (`app/models.py`)**: логика сравнения произношения, конвертация в пиньинь с тонами, генерация обратной связи (51%)
- **API эндпоинты (`app/main.py`)**: проверка произношения, статус моделей, корневой маршрут (96%)
- **Утилиты для тестирования**: загрузка и создание тестовых аудиофайлов (79-95%)
- **Инициализация приложения**: конфигурация CORS и startup events (100%)

## 4. Используемые технологии
- **pytest**, **pytest-cov** — запуск тестов и измерение покрытия
- **unittest.mock** — моки для ASR модели и файловой системы
- **FastAPI TestClient** — тестирование эндпоинтов
