1\. Выбранный проект



Проект: TheAlgorithms/Python



Ссылка на репозиторий:

https://github.com/TheAlgorithms/Python



Краткое описание:

Репозиторий содержит реализации различных алгоритмов на языке программирования Python.

Проект предназначен для изучения алгоритмов и структур данных и включает большое количество модулей, реализующих алгоритмы сортировки, поиска, работы со строками, динамического программирования и других разделов информатики.

3\. Добавленные тесты



Был создан дополнительный файл тестов:



test\_palidrome.py



Тесты были написаны для функции is\_palindrome из файла:



strings/palidrome.py



Данная функция проверяет, является ли строка палиндромом.



Были добавлены следующие проверки:

&#x20;• проверка строки, являющейся палиндромом

&#x20;• проверка строки, не являющейся палиндромом



Пример тестов:



from strings.palindrome import is\_palindrome
